# Teacher Guide

Day-to-day tasks for managing your gallery.

---

## Adding a New Class

1. Go to **Admin** → **New Class**
2. Enter class name (e.g., `Art 201 Spring 2026`)
3. Add teachers and students
4. Share the gallery URL with students

---

## Managing Students

### Add a Student
Admin → Select class → **Add Student** → Enter email

### Remove a Student
Admin → Select class → Find student → **Remove**

> Removing a student hides their work but doesn't delete it. You can re-add them later.

### Student Changed Classes?
Add them to the new class. They'll have separate collections in each class.

---

## Featuring Student Work

Showcase exceptional work on the gallery homepage.

1. Find the artwork in the gallery
2. Click the **Feature** button (star icon)
3. Featured work appears in the "Featured" section

To un-feature: Click the star icon again.

---

## Changing Themes

Customize how the gallery looks.

1. Go to **Admin** → **Themes**
2. Choose a preset or create custom colors
3. Apply to the whole gallery or specific classes

---

## Moderating Content

If you see inappropriate content:

1. Click the artwork
2. Click **Hide** (eye icon)
3. The artwork is hidden from everyone except you
4. Contact the student separately

To restore: Find in Admin → Hidden Items → **Restore**
