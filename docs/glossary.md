# Glossary

Terms used in this guide.

---

**Apps Script**
Google's tool for creating web apps. It runs on Google's servers and is free with Workspace.

**Archive**
Mark a class as inactive. Students can view but not upload. Used at end of semester.

**Class Gallery**
A gallery for one class/course. Contains collections for each enrolled student.

**Collection**
A student's portfolio within a class. Each student has one collection per class.

**Deploy**
Publish the web app so others can use it. Required after setup and after any code changes.

**Drive Folder ID**
The unique identifier for your Art Gallery folder. Found in the URL when inside the folder.

**Featured**
Work highlighted on the gallery homepage. Teachers select which artwork to feature.

**Google Workspace for Education**
The suite of Google tools (Drive, Sheets, etc.) provided free to schools.

**Profile**
A student's personal page showing all their work across all classes.

**Script Properties**
Configuration settings stored in your Apps Script project. Contains your IDs.

**Spreadsheet ID**
The unique identifier for your database spreadsheet. Found in the URL.

**Theme**
Visual settings (colors, layout) applied to the gallery.

**Web App URL**
The link people use to access your gallery. Looks like script.google.com/...
